## Crypto Arbitrage Scanner (Personal)

Запуск локально:

1. Создать виртуальное окружение (опционально)
2. Установить зависимости: `pip install -r requirements.txt`
3. Запустить сервер: `uvicorn app.main:app --host 0.0.0.0 --port 8000`
4. Открыть в браузере `http://localhost:8000/`

Функции:
- Скан стабильных спот-пар (USDT/USDC/FDUSD/BUSD/TUSD) по нескольким биржам через CCXT
- Учет торговых комиссий (taker) и фильтрация по ликвидности
- Вычисление gross/net спреда и приблизительной прибыли на заданный капитал

Примечания:
- Приложение для личного пользования, без регистрации/логина
- Список бирж и комиссии задаются в `app/main.py` (`DEFAULT_CONFIG`)
